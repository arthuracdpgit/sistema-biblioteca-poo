# Sistema de Biblioteca Digital - POO
Esta atividade foi desenvolvido para a disciplina de Programação Orientada a Objetos (POO) Prof. Daniel Bezerra na UNICAP.

## Tecnologias Utilizadas
* Java
* Git & GitHub

## Conceitos de POO Aplicados
* **Abstração:** Classes Usuario e Material.
* **Herança:** Especialização de Alunos, Professores e tipos de Materiais.
* **Interfaces:** Contratos para Exibição e Cálculo de Multas.
* **Polimorfismo:** Diferentes comportamentos para o resumo de cada item.

## Como Executar
Basta rodar a classe `Main.java` para ver a simulação do sistema no terminal.
